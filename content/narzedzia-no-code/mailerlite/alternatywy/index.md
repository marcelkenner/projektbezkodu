## Jak zacząć migrację — prosty plan (5–60 minut)
### Szybki plan na start
1. Eksport kontaktów z MailerLite (CSV) — sprawdź pola custom fields.  
2. Zmapuj pola u nowego dostawcy (tagi, pola niestandardowe).  
3. Przetestuj wysyłkę do małej próbki (100–500 kontaktów) i monitoruj dostarczalność.  
4. Przełącz formularze/landing page'y dopiero po udanym teście.

W praktyce: eksport i pierwsze testy to 5–30 minut; pełna migracja (segmenty, automatyzacje, integracje) to kilka dni roboczych, zależnie od złożoności integracji.

Jeśli nie wiesz, czy dany dostawca obsługuje twoje systemy (np. Shopify, WooCommerce, Zapier), sprawdź zakładkę integracje na stronie produktu — tam znajdziesz oficjalną listę. Jeśli integracja nie jest jawna, sprawdź dokumentację API lub zapytaj supportu dostawcy.
