# Useful Commands

- `source ~/.nvm/nvm.sh && npm install` – install dependencies.
- `source ~/.nvm/nvm.sh && npm run dev` – start Next.js dev server (App Router).
- `source ~/.nvm/nvm.sh && npm run build` – production build.
- `source ~/.nvm/nvm.sh && npm run start` – run build locally.
- `source ~/.nvm/nvm.sh && npm run lint` – run ESLint.
- `source ~/.nvm/nvm.sh && npm run tokens:build` – placeholder token build (replace with Style Dictionary workflow when ready).
- `pwsh ./scripts/scaffold.ps1 -Root ./content -JsonPath ./data/affiliate-programs.json -Domain "https://projektbezkodu.pl"` – scaffold markdown content (requires PowerShell 7).
- `pwsh ./scripts/build-evergreen.ps1 -Force` – run evergreen content builder script (requires PowerShell 7).
