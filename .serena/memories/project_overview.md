# Project Overview
- Purpose: blueprint and implementation scaffolding for multi-site affiliate stack combining Next.js frontend, Django API, Strapi CMS, and AWS infrastructure managed via Terraform.
- Key components: apps (frontend-next, backend-django, cms-strapi), infra modules and site-specific Terraform stacks, extensive playbooks/README for operations.
- Platforms/Services: AWS Amplify/CloudFront, ECS Fargate, ALB, RDS PostgreSQL, S3, Route 53, ACM, Secrets Manager, CloudWatch.
